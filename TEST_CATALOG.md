# Test Catalog

This document lists all automated tests and their intent.

Total tests: 46

## DtoClassLevelEdgeCasesTest.java

- `generateMapper_false_overridesOtherMapperOptions` - Verifies generateMapper=false prevents mapper generation even if update/toEntity are enabled.
- `mapper_crossPackage_usesQualifiedTypes` - Verifies cross-package mapper uses fully-qualified entity and DTO types.
- `typeUseAnnotations_areStrippedFromFieldTypes` - Verifies type-use annotations are stripped from generated DTO field types.
- `generateBuilder_respectsTargetsForMethodNames` - Verifies builder method names respect @IncludeInDto targets.

## DtoCopyValidationConfigTest.java

- `copyValidation_defaultFalse_doesNotCopyAnnotations` - Verifies default copyValidation=false does not copy validation annotations.
- `copyValidation_true_copiesJakartaAnnotations` - Verifies copyValidation=true copies jakarta.validation annotations by default.
- `validationNamespace_javax_copiesOnlyJavaxAnnotations` - Verifies validationNamespace=JAVAX copies javax.validation annotations (and not jakarta).
- `copyValidation_perFieldOverride_disablesForTargetDto` - Verifies per-field copyValidation override can disable copying for a specific DTO.
- `copyValidation_overrideLengthMismatch_failsCompilation` - Verifies per-field copyValidation override length mismatch triggers compilation error.

- `includeInDto_genericTypes_arePreserved` - Verifies generic field types are preserved in the generated DTO.
